# Ecommerce App

## Descripción
🌟 Esta aplicación de comercio electrónico permite a los usuarios comprar y vender productos de manera efectiva y fácil. 

## Características
* 🛒 Carrito de compras
* 🔍 Búsqueda de productos
* 🏷️ Gestionar productos
* 💳 Métodos de pago seguros

## Instrucciones de Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/Marlon4789/Ecommerce-App.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd Ecommerce-App
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```

## Estructura del Proyecto
| Carpeta/Archivo       | Descripción                      |
|-----------------------|----------------------------------|
| `src/`                | Contiene los archivos de código. |
| `public/`             | Archivos estáticos y index.html. |
| `README.md`           | Documentación del proyecto.      |

## Uso
Para iniciar la aplicación, usa el siguiente comando:
```bash
npm start
```
